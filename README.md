# LDACA_notebooks
A GitHub repository for the LDACA notebooks developed by the QUT Digital Observatory as companion notebooks for the AusReddit and Twittersphere datasets

exploration.ipynb - notebook for exploring the Twittersphere dataset
exploration_notebook.py - code base to drive the exploration notebook

exploration_ar.ipynb - notebook for exploring the AusReddit dataset
exploration_notebook_ar - code base to drive the exploration_ar notebook

requirements.txt - python package requirements for notebooks

Note: remember to load the data from the correct path, i.e. where you downloaded/extracted the data to (the data folder is a good place)
